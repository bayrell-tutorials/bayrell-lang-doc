Язык программирования BAYRELL Language

Bayrell Language - это транслятор языка. На сегодняшний день работает трансляция из BAYRELL Language в php, nodejs, javascript.

Технические возможности:

  • Транслятор языка в php, nodejs, javascript
  • Встроенный в язык HTML шаблонизатор
  • Глобальный массив Entity
  • Точки входа @Route
  • Бэкенд api
  • Вызов Api из Frontend
  • Функциональное программирование
  • Неизменяемые структуры данных
  • Функциональные цепочки вызовов функций (pipe)
  • Асинхронное программирование async/await
  • Контекст
  • Кэширование результатов функция, через параметр memorize
  • Лямбда цепочки
